## Informacje ogólne
Projekt umożliwiający wizualizację działania b-drzewa.  
Program pozwala na przeprowadzenie następujących operacji:
  * wstawianie 
  * wyszukiwanie 
  * usuwanie 
  * modyfikacja / elementów b-drzewa
  * równoważenie b-drzewa
